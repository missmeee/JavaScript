# 表单

使用form标签来创建一个表单



form的属性：

action 表单要提交的服务器地址



文本框

<input type="text" name="username">

注意：数据要提交到服务器中，必须要为元素指定一个name



密码框

<input type="password" name="password">



提交按钮

<input type="submit" value="注册">





autocomplete="off"  关闭自动补全

readonly 将表单项设置为只读

disabled 将表单项设置为禁用

autofocus 设置表单项自动获取焦点
